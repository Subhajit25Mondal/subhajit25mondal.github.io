Introduction

This project is a clone of the depreciated Google URL Shortener(goo.gl). I have used PHP in the previous version of this project. But now I am completely migrating this project to NodeJS. Thus, this project is currently under development.

Technology Stack

NodeJS, Express, MongoDB, React, NextJS and Bulma frontend framework
